# 文件工坊 / File Toolbox

多功能文件处理工具 - 图片压缩/格式转换/PDF合并拆分/二维码生成/文本处理/格式转换

- 纯本地浏览器处理，不上传任何数据到第三方
- 支持批量操作
- 无需安装，打开即用

## 功能

| 模块 | 功能 |
|------|------|
| 🖼️ 图片处理 | 压缩、格式转换(PNG/JPG/WebP/BMP/ICO)、缩放 |
| 📄 PDF工具 | 合并、拆分 |
| 🔗 二维码 | 单次生成、批量生成 |
| 📝 文本处理 | 去重、排序、查找替换(支持正则) |
| 📊 格式转换 | CSV↔Excel |
